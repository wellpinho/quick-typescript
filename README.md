## Classe
Abstração do que há em comum no mundo real e trazer para programação.

1. classe tem: __nome__, __atributos__ e __métodos__
```
class Mamifero {
  data de nascimento
  peso
  alimento

  comer()
  mover()
  mamar()
}
```
```
class Empresa {
  nome
  CNPJ
  funcionarios[]

  contratar()
  demitir()
}
```
```
class Funcionario {
  nome
  CPF
  salario

  trabalhar()
  baterPonto()
}
```
